# API Endpoints and Functions

Base URL: https://hbp-api.onrender.com

## Authentication
1. **POST** `/api/auth/register`  
   - Register a new user  
   - Body: `{ email, username, password, firstName, lastName }`

2. **POST** `/api/auth/login`  
   - Login a user  
   - Body: `{ email, password }`

3. **GET** `/api/auth/profile`  
   - Get the logged-in user's profile  
   - Headers: `{ Authorization: "Bearer <token>" }`

---

## Articles
1. **GET** `/api/articles`  
   - Get all published articles (public access)

2. **GET** `/api/articles/:id`  
   - Get a specific article by ID

3. **POST** `/api/articles`  
   - Create a new article  
   - Headers: `{ Authorization: "Bearer <token>" }`  
   - Body: `{ title, content, status }`

4. **PUT** `/api/articles/:id`  
   - Update an article  
   - Headers: `{ Authorization: "Bearer <token>" }`  
   - Body: `{ title, content, status }`

5. **DELETE** `/api/articles/:id`  
   - Delete an article  
   - Headers: `{ Authorization: "Bearer <token>" }`

6. **POST** `/api/articles/:id/like`  
   - Like an article  
   - Headers: `{ Authorization: "Bearer <token>" }`

7. **DELETE** `/api/articles/:id/like`  
   - Unlike an article  
   - Headers: `{ Authorization: "Bearer <token>" }`

---

## Comments
1. **GET** `/api/comments/:articleId`  
   - Get all comments for an article

2. **POST** `/api/comments/:articleId`  
   - Add a comment to an article  
   - Headers: `{ Authorization: "Bearer <token>" }`  
   - Body: `{ content }`

3. **PUT** `/api/comments/:id`  
   - Edit a comment  
   - Headers: `{ Authorization: "Bearer <token>" }`  
   - Body: `{ content }`

4. **DELETE** `/api/comments/:id`  
   - Delete a comment  
   - Headers: `{ Authorization: "Bearer <token>" }`

---

## Users
1. **GET** `/api/users`  
   - Get a list of all users (Admin only)  
   - Headers: `{ Authorization: "Bearer <token>" }`

2. **GET** `/api/users/:id`  
   - Get a specific user's profile

3. **PUT** `/api/users/:id`  
   - Update a user's profile (Admin only)  
   - Headers: `{ Authorization: "Bearer <token>" }`  
   - Body: `{ email, username, firstName, lastName, role }`

4. **DELETE** `/api/users/:id`  
   - Delete a user (Admin only)  
   - Headers: `{ Authorization: "Bearer <token>" }`
